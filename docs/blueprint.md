# **App Name**: Qreator Kids

## Core Features:

- Single QR Code Generation: Generate a single QR code from user-provided data with options for customization.
- Multiple QR Code Generation: Generate multiple QR codes from data separated by line breaks, for batch creation.
- Customization Options: Allow users to customize padding, border, size, foreground color, and background color of the QR codes.
- Image Format Selection: Let users choose the output image format for the QR codes (e.g., PNG, JPEG, SVG).
- Bulk Download as ZIP: Enable downloading multiple generated QR codes as a single ZIP archive.
- Theme Selection: Implement light, dark, and auto themes for the website based on user preference or system settings.

## Style Guidelines:

- Primary color: Vibrant yellow (#FFDA63) to convey a playful, kid-friendly feel.
- Background color: Soft, desaturated yellow (#FAF2DD) to maintain brightness without overwhelming the user.
- Accent color: Light orange (#FFB347) for interactive elements to catch the eye, enhancing usability.
- Body and headline font: 'PT Sans', a humanist sans-serif to create a friendly modern style for kids and parents alike.
- Utilize friendly, rounded icons suitable for children. Focus on simplicity and easy recognition.
- Employ a clean and straightforward layout with ample spacing for easy navigation, keeping it accessible and pleasing for users of all ages.
- Use soft, gentle animations for interactions to give a delightful and responsive user experience, reinforcing the app's engaging nature.
