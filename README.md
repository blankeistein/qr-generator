## Installation

Pastikan Anda memiliki Node.js dan npm/yarn terinstal.

- Kloning Repositori:

```bash
  git clone https://github.com/blankeistein/qr-generator.git
  cd qr-generator
```

- Pasang Dependensi:
```bash
  npm install 
```

- Jalankan Aplikasi:
```bash
  npm run dev atau yarn dev
```
